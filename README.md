# STJDNet

This repository will provide the official implementation of **STJDNet**, a skeleton-based network for two-person interaction recognition.

## Code Availability

The code is currently being organized. It will be made publicly available upon acceptance of the paper.

## Overview

STJDNet is developed for skeleton-based two-person interaction recognition. The model is evaluated on the two-person interaction subsets of NTU RGB+D and NTU RGB+D 120 under standard evaluation protocols.

## Planned Contents

After release, this repository will include:

- Source code of STJDNet
- Data preprocessing scripts
- Training and evaluation scripts
- Configuration files
- Instructions for reproducing the reported results
